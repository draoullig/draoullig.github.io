Apple propose l'Apple Push Notification Service (APN). Ce service permet à un terminal Ipad/Iphone d'être constamment connecté au serveur d'Apple push notification. Lorsque vous voulez envoyer une notification push à une application installée sur un terminal distant, vous (le fournisseur) envoyez vos messages via l'APN afin que ce dernier puisse délivrer le message push à l'application cible installée sur le média distant.

Dans cette série de page, nous allons dérouler les différentes étapes nécessaires pour créer une application iPhone qui utilise l'APN et l'implémentation serveur java pour envoyer des messages.

  1. Demander une demande de certificat à une autorité de certificat
  2. Demander un App ID
  3. Configurer l'App ID pour la notification Push
  4. Création d'un Provisioning Profile
  5. Création de l'application Iphone
  6. Création d'un certificat P12
  7. Création de la partie serveur de notification Push
  8. Test des applications
