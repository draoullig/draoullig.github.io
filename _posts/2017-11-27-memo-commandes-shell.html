  • Se logger en root avec son environnement (lorsque que l'on est sudoer) : sudo -i

  • Supprimer une ligne dont on connait le numèro dans un fichier non binaire (ex: 33ème ligne) : sed -i -e '33d' nom_du_fichier

  • Connaître les différentes ip associées aux interfaces sous linux : ip a

  • Remplacer une chaine dans plusieurs fichiers avec grep et sed sous linux : grep -rl matchstring somedir/ | xargs sed -i 's/string1/string2/g'

  • Connaître l'état des connexions sur une machine linux : netstat -anotp

  • Escaper une variable dans un script shell : variable_escaped=$(echo "${variable}" | sed -e 's/[\/&]/\\&/g')
