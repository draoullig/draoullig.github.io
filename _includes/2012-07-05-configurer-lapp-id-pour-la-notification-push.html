1. Pour effectuer cette opération, cliquer sur le lien Configure qui se trouve à droite dans la description de l'App ID.

  2. Une nouvelle page s'ouvre.

  3. Cocher la case Enable for Apple Push Notification service puis cliquer sur le bouton Configure pour Development Push SSL Certificate. Une popup s'ouvre.

  4. Cliquer sur Continue. Sélectionner le fichier de demande de certificat généré au début du processus via le bouton. Cliquer sur Generate.
  5. Le certificat est généré. Cliquer sur Continue.

  6. Cliquer sur Download. Puis sur le bouton Done.
  7. Le fichier généré s'appelle aps_developpement.cer et contient le certificat ssl. Double cliquer sur le fichier, ce dernier est ajouter au Trousseaux d'accès.
