La première étape consiste à générer une demande de certificat au format fichier pour pouvoir effectuer une demande de certificat SSL de développement dans un deuxième temps.

  1. Lancer l'application trousseaux d'accès sous MAC OS X
  2. Ouvrir le menu Trousseaux d'accès/Assistant de certification/Demander un certificat à une autorité de certificat...

  3. Saisir les informations demandées et enregistrer la demande sur le disque. Cliquer sur Continuer.

  4. Sauvegarder la demande de certificat en gardant les valeurs par défaut en cliquant sur Enregistrer.

  5. Sur la fenêtre suivante cliquer sur Terminé.
