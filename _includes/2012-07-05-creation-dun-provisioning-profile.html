L'étape suivante consiste à créer un provisioning profile pour permettre l'installation du logiciel sur un terminal.

  1. Cliquer sur l'item Provisionning du menu de gauche. Une nouvelle page s'ouvre.

  2. Cliquer sur New Profile.
  3. Dans la nouvelle page saisir Developpement_Test_alerte dans Profil Name, cocher la case Certificates, sélectionner TestAlerte dans App ID et cocher les terminaux que vous souhaitez ajouter à ce profil de provisionning. Cliquer sur Submit.

  4. Le nouveau certificat est généré et apparaît dans la liste.
