Chaque application iPhone qui utilise le service APN doit avoir un ID d'application unique. Nous allons donc créer cet ID d'application pour la notification Push. Attention, il n'est pas possible de supprimer des App ID dans l'interface de gestion Apple, toute création est définitive.

  1. S'identifier sur le portail iPhone Developer Program (http://developer.apple.com/iphone/) puis cliquer sur IOS Provisionning Portal dans le menu de droit.

  2. La page du IOS Provisioning Portal s'ouvre.

  3. Cliquer sur l'item App Ids du menu de gauche, puis sur le bouton New App ID.

  4. Entrer TestAlerteAppID dans le champs Description.
  5. Sélectionner Generate New ou Use Team ID dans le champs Bundle Seed ID (App ID Prefix) si vous avez déjà un identifiant déclaré.
  6. Entrer un identifiant (Ex : com.fortuneo.TestAlerte) dans le champs Bundle Identifier (App ID Suffix),puis cliquer sur Submit.

  7. Normalement la nouvelle entrée doit apparaître dans la liste.
